- 入口与路由：单文件 `src/index.js` 导出 `fetch` 处理函数，统一处理 GET/POST/OPTIONS 请求并返回 CORS 头。
- 请求解析：支持 JSON POST 与 URL 参数 GET 两种输入形态，校验 `html` 必填并合并默认配置。
- 渲染集成：调用 Cloudflare Browser Rendering API (`/browser-rendering/screenshot`)，通过环境变量注入 `CLOUDFLARE_ACCOUNT_ID` 与 `CLOUDFLARE_API_TOKEN`。
- 响应策略：支持直接返回图片流或包含 Base64 数据的 JSON 响应，由 `response_type` 参数控制。
- 配置管理：通过 `wrangler.toml` 定义兼容性标志 `nodejs_compat_v2` 及默认视口尺寸，敏感凭据通过 `wrangler secret` 管理。
