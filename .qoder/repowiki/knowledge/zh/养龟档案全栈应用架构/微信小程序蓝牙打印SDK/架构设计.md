- **核心架构**：采用适配器模式（Adapter Pattern），通过 `lib/adapter/BleAdapter` 将底层微信原生 BLE API (`wx.openBluetoothAdapter`, `wx.createBLEConnection` 等) 抽象为统一的 `IBleAdapter` 接口，实现平台无关性。
- **入口设计**：`LPAPIFactory` 作为单例工厂（`lib/index.d.ts`），负责初始化 SDK 实例并注入 Canvas 上下文（支持离屏或隐藏 Canvas），协调绘图与打印任务。
- **依赖关系**：模块依赖于核心逻辑包 `lpapi-ble`，本模块 (`lpapi-ble-wx`) 仅负责微信小程序环境的特定实现与桥接。
- **测试结构**：`test/lpapi-ble-test` 包含完整的微信小程序 Demo，涵盖文本、条码、二维码、图片、表格等多种打印场景的页面实现 (`pages/*`)。
