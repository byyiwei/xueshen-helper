- **前端-后端通信**：小程序端通过 `wx.cloud.callFunction` 统一调用云函数（如 `pet`, `record`, `login`），采用 `{ action, data }` 的请求契约。
- **跨域渲染链路**：前端将 HTML 模板发送至 `html2image` 云函数，该函数作为网关转发至外部渲染服务（本地 Puppeteer 服务器或 Cloudflare Worker），实现动态内容转图。
- **共享安全边界**：前后端共用 `securityChecker` 模块，前端负责上传触发异步审核，后端/云函数负责调用微信官方 API 并记录日志至 `security_logs` 集合。
- **硬件交互隔离**：蓝牙打印逻辑封装在 `lpapi` SDK 中，通过 `printer.js` 工具类在小程序端统一管理设备连接与标签绘制，与云端业务逻辑解耦。
