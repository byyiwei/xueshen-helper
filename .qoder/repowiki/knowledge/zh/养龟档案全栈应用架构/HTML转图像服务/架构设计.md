- **入口与核心**：`server.js` 作为单一入口，使用 Node.js 原生 `http` 模块构建服务器，集成 `puppeteer` 进行浏览器自动化渲染。
- **配置管理**：`config.js` 实现多层级配置加载（环境变量 H2I_ > config.json > 硬编码默认值），支持嵌套键映射和类型自动转换。
- **资源管理**：采用单例浏览器实例池模式，通过 `getBrowser` 懒加载 Chromium，并在断开连接时自动重置；页面实例在每次请求后显式关闭以释放内存。
- **日志系统**：`logger.js` 提供轻量级双通道日志（控制台彩色输出 + 按日切分的文件追加），封装了 HTTP 请求生命周期追踪方法。
- **接口边界**：暴露 `/health`、`/api-docs`、`/config` 等管理端点，主转换逻辑通过 `POST /` 接收 JSON 载荷并返回 Base64 编码图像。
