## 🔧 两个版本的核心差异

### 1. @require 外部脚本来源

**Greasy Fork 版本 (xueshen-gf.js)**：
```
// @require  https://cdn.jsdelivr.net/npm/vue@3.4.31/dist/vue.global.prod.js
// @require  https://cdn.jsdelivr.net/npm/pinia@2.1.7/dist/pinia.iife.min.js
// ... 全部改为 jsDelivr CDN
```
Greasy Fork 只允许 jsDelivr、cdnjs、Google Hosted Libraries 等白名单 CDN，不允许自建服务器。

**脚本猫版本 (xueshen-sc.js)**：
```
// @require  https://xs.openget.cn/libs/vue/3.4.31/vue.global.prod.js
// ... 保持自有服务器
```
脚本猫无此限制，可使用任意 CDN。

**Greasy Fork 版本**：

## ⚠️ Greasy Fork 审核注意事项

### 可能仍被拒绝的风险点

1. **代码格式问题**
   - 当前代码经过 webpack 打包，有部分压缩（如第55行 CSS 压缩成一行）
   - Greasy Fork 要求「代码不得混淆或压缩」
   - **应对方案**：如果被拒，需向审核员说明这是 webpack 非压缩输出，或考虑将 CSS 提取出来格式化

2. **@antifeature payment**
   - 已正确标注付费功能
   - 描述已明确「基础刷课功能永久免费」

3. **脚本描述与功能一致性**
   - Greasy Fork 要求「脚本必须提供描述来说明此脚本会做什么」
   - 当前 description 已涵盖主要功能

4. **data: URI @require**
   - `@require data:application/javascript,window.Vue%3DVue%3B`
   - 这是 data URI，不是外部脚本，理论上应该允许

### 提交 Greasy Fork 时的建议

1. **描述填写**：使用 `publish_materials/脚本简介-油猴脚本猫.md` 的内容
2. **分类选择**：选择「其他」或「学习辅助」
3. **敏感内容**：勾选「包含付费内容或服务」
4. **如被拒绝**：
   - 检查拒绝原因
   - 如果是代码格式问题，尝试格式化压缩的 CSS
   - 如果是 CDN 问题，确认所有 @require 都是 jsDelivr
   - 可以在申诉中说明脚本的合法用途

1. 访问 https://greasyfork.org/
2. 注册/登录
3. 点击「发布脚本」
