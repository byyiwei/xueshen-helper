   ```

   ```
3. **@contributionURL**：可选，添加捐赠/赞助链接
4. **@description 精简**：当前描述较长，油猴列表展示时可能被截断。建议控制在120字以内，核心卖点放前面：

## 📱 各平台发布指南

### 一、Greasy Fork（油猴叉）发布

**官网**：https://greasyfork.org/

#### 发布步骤：
1. 注册/登录 Greasy Fork 账号
2. 点击右上角「发布你编写的脚本」
3. 填写脚本信息：
   - **脚本名称**：`🚀学神助手｜超星·智慧树全能刷课`
   - **脚本描述**：使用精简版description（上面建议的版本）
   - **脚本内容**：粘贴 `xueshen.js` 完整代码
   - **附加信息 → 版本号**：5.0.0
   - **附加信息 → 许可证**：选择「保留所有权利」或其他
   - **附加信息 → 分类**：其他
   - **敏感内容标记**：勾选「包含付费内容或服务」（对应 @antifeature payment）
4. 截图/图片：上传 `images/hero_clean.jpg` 作为主截图
5. 详细说明：粘贴 `脚本简介-油猴脚本猫.md` 的内容（Greasy Fork 支持 Markdown）
6. 点击「发布」

#### Greasy Fork 注意事项：
- Greasy Fork 对脚本描述和代码有审核机制，首次发布可能需要等待审核
- 不要在描述中放置过多外部链接（可能被判为垃圾脚本）
- 确保 `@antifeature` 标注正确，付费功能必须声明
- 代码中不要包含混淆/加密代码（Greasy Fork 禁止）
- 每次更新版本号必须递增

### 二、脚本猫（ScriptCat）发布

**官网**：https://scriptcat.org/

#### 发布步骤：
1. 注册/登录脚本猫账号
2. 点击「发布脚本」
3. 填写脚本信息：
   - **脚本名称**：`🚀学神助手｜超星·智慧树全能刷课`
   - **脚本描述**：同Greasy Fork
   - **脚本代码**：粘贴 `xueshen.js` 完整代码
   - **版本**：5.0.0
   - **分类**：学习辅助/工具类
   - **标签**：自动刷课、AI答题、超星学习通、智慧树、网课
4. 封面图：上传 `images/hero_clean.jpg`
5. 脚本详情：粘贴 `脚本简介-油猴脚本猫.md` 内容
6. 点击「提交审核」

#### 脚本猫注意事项：
- 脚本猫是国内平台，访问速度快，用户群体主要是国内学生
- 审核相对宽松，但也需要等待
- 支持评论互动和评分
- 可以设置脚本为「首发脚本猫」获得更多曝光
- 脚本猫支持订阅机制，可以引导用户订阅更新通知

## 📝 产品展示页部署

`index.html` 是一个完整的产品介绍单页网站，可以部署到服务器上作为脚本的官方介绍页：

1. 将 `publish_materials/` 目录下的所有文件上传到服务器
2. 配置Nginx指向该目录，或放到已有的站点下
3. 在脚本元数据中添加 `@homepageURL` 指向该页面
4. 在平台发布时将首页链接放在脚本详情中

建议部署路径：`https://xs.openget.cn/intro/` 或直接作为根目录首页。

## 🔄 更新发布流程

每次更新版本时的发布流程：

1. 更新 `xueshen.js` 中的 `@version` 版本号
2. 在脚本元数据下方或代码中更新更新日志
3. 在Greasy Fork和脚本猫后台更新脚本代码
4. 在「其他信息→更新日志」中填写本次更新内容
5. 如为重大更新，可在平台发布更新公告
6. 在用户群/社群中通知用户更新
