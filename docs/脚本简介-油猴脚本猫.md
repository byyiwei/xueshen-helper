# 🚀学神助手｜学习通｜超星·智慧树全能刷课

## 📖 脚本简介

学神助手是一款专为大学生打造的网课辅助工具，支持超星学习通、智慧树等主流网课平台，集成AI智能辅助学习、视频倍速播放、自动切换任务点、作业考试全自动完成等功能，安装即用，解放双手，让网课不再占用你的宝贵时间。

---

## ✨ 核心功能

### 🤖 AI智能辅助学习
- **DeepSeek大模型驱动**：接入DeepSeek-V3和DeepSeek-R1双模型，推理能力更强
- **题库+AI双重保障**：海量题库优先搜索，题库盲区AI秒补，精准无死角
- **全题型覆盖**：支持单选、多选、判断、填空等多种题型
- **本地缓存**：答过的题目自动缓存，不重复消耗次数

### ⚡ 极速秒过视频
- **倍速播放**：支持1-16倍速自由调节，建议1-2倍稳定使用
- **自动连播**：视频播放完毕自动进入下一集/下一章节
- **弹题处理**：自动识别视频中弹题并跳过处理
- **字体破解**：一键破解超星字体加密，正确显示题目内容

### 🎯 全程自动托管
- **自动切换章节**：自动识别并切换到下一个未完成的任务点
- **章节测验**：自动完成章节测验并提交
- **作业自动答**：自动完成作业，满足设定正确率后自动提交
- **考试全自动**：支持整卷预览模式，自动答题、自动交卷

### 📚 阅读类任务自动完成
- 图书自动翻页阅读
- 文档/PPT自动浏览
- 直播任务自动挂机
- 阅读时长自动累积

### 🛡️ 安全稳定
- 模拟真实人工操作行为
- 正确率阈值控制，低于设定值不提交
- 多节点自动切换，保障服务稳定
- 日志定时自动清理，保护隐私安全

## 📦 安装教程

### 第一步：安装脚本管理器
首先需要在浏览器中安装 **Tampermonkey（油猴）** 扩展：
- **Chrome/Edge**：在Chrome应用商店/Edge外接程序商店搜索 "Tampermonkey" 安装
- **Firefox**：在附加组件商店搜索 "Tampermonkey" 安装
- 油猴官网：[https://www.tampermonkey.net/](https://www.tampermonkey.net/)

### 第二步：安装本脚本
安装油猴后，点击本页面的「安装此脚本」按钮，在弹出的油猴安装页面点击「安装」即可。

### 第三步：开始使用
1. 安装完成后，打开超星学习通或智慧树的课程页面
2. 页面左上角会出现学神助手的悬浮控制面板
3. 进入课程视频/作业/考试页面，脚本会自动开始运行
4. 建议先在「设置」面板中调整好倍速和答题选项

## ⚙️ 功能设置说明

| 设置项 | 说明 | 建议值 |
|--------|------|--------|
| 答题模式 | 纯题库/纯AI/题库+AI | 题库+AI（推荐） |
| 视频倍速 | 视频播放速度 | 1-2倍（过高有风险） |
| 正确率阈值 | 达到此正确率才自动提交 | 80% |
| 自动提交 | 完成后自动提交作业/章节 | 开启 |
| 自动切换章节 | 完成当前任务后自动切换 | 开启 |
| 自动答题 | 进入题目页面自动开始答题 | 刷课开启，手动答题关闭 |
| 自动交卷 | 考试答完后自动交卷 | 根据需要设置 |

## ❓ 常见问题

**Q: 使用脚本会被检测到吗？**
A: 脚本模拟人工操作，内置防检测机制。但任何工具都不能保证100%安全，建议倍速1-2倍，不要长时间挂机。

**Q: 为什么视频看完了任务点没完成？**
A: 部分课程有防倍速机制，倍速过高可能导致进度不记录。建议调低倍速至1-1.5倍，刷新页面重新观看。

**Q: AI答题准确率怎么样？**
A: 客观题（选择、判断）准确率较高，主观题会给出参考答案。题库命中率约85%，题库+AI整体准确率90%以上。

**Q: 脚本面板不显示怎么办？**
A: 检查油猴扩展是否启用、脚本是否启用、是否在支持的平台页面上，尝试刷新页面或清除缓存。

**Q: 支持哪些浏览器？**
A: 推荐Chrome、Edge、Brave等Chromium内核浏览器，Firefox也可使用。不支持IE和Safari。
