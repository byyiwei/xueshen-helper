## 主要优化点：

1. **@description**：精简到适合列表展示的长度，核心卖点前置
2. **@description:zh-CN**：添加中文描述，确保中文平台正确显示
3. **@homepageURL**：添加脚本主页链接
4. **@supportURL**：添加用户支持页面链接
5. **@icon**：建议改为自有域名的favicon
6. **@connect**：清理冗余域名，保留必要的连接白名单
7. **@antifeature**：更详细的付费功能说明
8. **@tag**：增加「刷课」标签，提高搜索命中率

## 需要确认的事项：
- [ ] `@icon` 的favicon是否已上传到服务器
- [ ] `@homepageURL` 和 `@supportURL` 对应的页面是否可访问
- [ ] `@connect` 中是否需要保留其他域名（如第三方题库接口域名）
